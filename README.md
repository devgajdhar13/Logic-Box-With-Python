# 🧠 Logic Box

A simple Python console application that provides two features:
- ⭐ Generate a star (`*`) pattern
- 🔢 Analyze a range of numbers (Even/Odd and Sum)

## ✨ Features
- 🌟 Generates a right-aligned triangle pattern.
- 🔍 Identifies whether each number in a given range is Even or Odd.
- ➕ Calculates the sum of all numbers in the selected range.
- 🚪 Menu-driven interface with an Exit option.
